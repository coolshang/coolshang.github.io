git入门

使用git先要安装git.exe客户端
这里只讲解Windows上安装git
下载地址https://git-scm.com/downloads
安装好后打开git Bash.
接着输入：
git –version
git version 2.9.2.windows.1

因为git是分布式版本控制系统,所以你得为自己的机器设置一个名称与eamil。也许你会担心如果有人冒充了怎么办?这个不必担心,真的有冒充的也是有办法可查的。
